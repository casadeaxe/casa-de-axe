CASA DE AXÉ - SITE DE E-COMMERCE

Este é o site da loja Casa de Axé.

Como usar:
1. Acesse index.html para a página inicial.
2. Vá até "Produtos" para ver e adicionar produtos ao carrinho.
3. O "Carrinho" mostra os itens selecionados e simula uma compra.
4. "Contato" tem um formulário simples para mensagens.

Este site é estático e pode ser publicado gratuitamente com GitHub Pages.

Para integrar pagamentos reais (Pix, Mercado Pago, etc), será necessário back-end.
